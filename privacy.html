Integritetspolicy

Senast uppdaterad: 14 april 2026

1. Personuppgiftsansvarig

LIA-portalen (liaportalen) är personuppgiftsansvarig för behandlingen av personuppgifter på denna webbplats. Har du frågor om hur vi hanterar dina uppgifter är du välkommen att kontakta oss via e-post på kontakt@liaportalen.se.

2. Vilka personuppgifter behandlar vi?

2.1 Företag som publicerar LIA-annonser

När ett företag fyller i annonsformuläret samlar vi in:

  • Företagsnamn
  • E-postadress för ansökan

Dessa uppgifter lagras enbart i din webbläsares lokala lagring (localStorage) på din egen dator. De skickas aldrig till någon server som vi driver. Uppgifterna visas publikt i annonsen så att studenter kan ta kontakt.

2.2 Studenter som besöker sajten

Vi samlar inte in några personuppgifter om studenter som söker bland annonserna. Ingen inloggning eller registrering krävs.

3. Ändamål och rättslig grund

Ändamål Uppgifter Rättslig grund
Visa LIA-annonser med kontaktmöjlighet E-postadress, företagsnamn Berättigat intresse (GDPR art. 6.1 f) – möjliggöra kontakt mellan student och arbetsgivare

4. Lagringstid

Uppgifter som lagras i webbläsarens localStorage finns kvar tills du manuellt rensar webbläsardata, eller tills du tar bort annonsen. Ingen automatisk gallring sker från vår sida eftersom vi inte har tillgång till dessa uppgifter.

5. Tredjeparter

Webbplatsen kommunicerar med följande externa tjänster. Din IP-adress och webbläsarinformation kan komma att behandlas av dessa parter när du besöker sajten.

Tjänst Ändamål Integritetspolicy
GitHub Pages (Microsoft) Webbhotell – serverar webbplatsens filer github.com/privacy
Arbetsförmedlingen JobSearch API Hämtar lediga LIA-annonser från Arbetsförmedlingen arbetsformedlingen.se/integritet
Jobbsafari.se Annonser hämtas periodiskt av en server och cachas lokalt – inga anrop görs från din webbläsare jobbsafari.se/integritetspolicy

Vi använder inga spårningscookies, reklamkakor eller analysverktyg (som Google Analytics).

6. Cookies och lokal lagring

Webbplatsen sätter inga cookies. Vi använder webbläsarens localStorage uteslutande för att tillfälligt spara annonser som du publicerat via formuläret, så att de syns kvar nästa gång du öppnar sajten på samma dator. Denna lagring är helt lokal och skickas inte till oss.

7. Dina rättigheter

Enligt GDPR har du rätt att:

  • Få tillgång till de uppgifter vi behandlar om dig
  • Rätta felaktiga uppgifter
  • Radera dina uppgifter ("rätten att bli glömd")
  • Invända mot behandling baserad på berättigat intresse
  • Inge klagomål till Integritetsskyddsmyndigheten (IMY) på imy.se

Eftersom annonsdata enbart lagras i din egen webbläsare kan du radera dina uppgifter när som helst genom att rensa webbläsarens localStorage. Kontakta oss på kontakt@liaportalen.se vid övriga frågor.

8. Ändringar i denna policy

Vid väsentliga ändringar uppdateras datumet högst upp på denna sida. Vi rekommenderar att du läser igenom policyn vid jämna mellanrum.
